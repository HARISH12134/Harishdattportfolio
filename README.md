# Harish M - UI/UX Designer Portfolio

A stunning portfolio website featuring liquid glass aesthetics and interactive project showcases.

## 🎨 Features

- **Liquid Glass Design** - Modern glassmorphism UI with smooth animations
- **Interactive Projects** - 6 featured projects with screenshot galleries
- **Responsive Layout** - Optimized for all devices
- **Smooth Animations** - Motion-powered transitions and effects
- **Project Modals** - Interactive carousels with Figma links

## 🛠️ Tech Stack

- React 18
- Vite
- Tailwind CSS v4
- Motion (Framer Motion)
- Lucide React Icons

## 🚀 Quick Start

### Install Dependencies
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

## 📦 Deployment

### Vercel (Recommended)
1. Push code to GitHub
2. Import repository on [vercel.com](https://vercel.com)
3. Deploy automatically

### Netlify
1. Build the project: `npm run build`
2. Deploy the `dist` folder to [netlify.com](https://netlify.com)

### GitHub Pages
1. Install: `npm install --save-dev gh-pages`
2. Add to package.json:
   ```json
   "homepage": "https://YOUR_USERNAME.github.io/REPO_NAME",
   "scripts": {
     "predeploy": "npm run build",
     "deploy": "gh-pages -d dist"
   }
   ```
3. Deploy: `npm run deploy`
